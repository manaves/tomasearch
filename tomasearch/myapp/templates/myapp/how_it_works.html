How it works

üîç Search

You can perform four types of searches to explore the tomato database:

1. Search by gene

Enter one or more genes (Solyc00g000000.0), separated by commas. Select the columns to display:

  • GO identifier
  • GO term
  • Subontology
  • Subgraph

2. Search by GO identifier

Enter one or more GO IDs (GO:0000000), separated by commas. Optional columns:

  • GO term
  • Subontology
  • Subgraph

3. Search by GO term

Search ontology terms (e.g., photosynthesis, membrane). Partial matches are included. Optional columns:

  • GO identifier
  • Subontology
  • Subgraph

4. Search by subontology

Filter by one of:

  • Cellular component
  • Biological process
  • Molecular function

If you select GO identifier and Subgraph, enrichment and p-value data will be shown (threshold: 2.601457e-05).
